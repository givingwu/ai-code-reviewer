# AI-Code-Reviewer

This is a forked project from [dify-gitlab-mr-reviewer](https://github.com/zgldh/dify-gitlab-mr-reviewer) by [zgldh](https://github.com/zgldh).

## Features

- 集成[前端规范](https://mountainwu.notion.site/296d0703435c4d8686f6f84b44bb06a3?pvs=74)知识库　
- 集成 TV Bot 机器人实时推送 AI Code Review Message
- 集成 Gitlab Webhook 发送消息
